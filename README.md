# dotfiles

## Dependencies

+ Only Linux
+ Git installed

## Instalation

```bash
git clone https://github.com/flaviareisv/dotfiles.git $HOME/dotfiles
cd dotfiles
./dotfile
```
Launch `vim` and run `:PluginInstall`
